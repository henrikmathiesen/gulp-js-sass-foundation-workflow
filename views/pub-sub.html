Pub Sub

Following the Youtube course, LearnCodeAcademys "Modular Javascript #4 - PubSub Javascript Design Pattern" the goal is to register custom events and let other parts of the javascript code to subscribe and act on the events.

When user adds a name, it is saved to an array

We have two listeners, js files, who are interested in this. One of them has the responsibility to add a red outline to the field, the outher has the responsibility to add a green outline to the button. The outline will be thicker depending on how many items the list contains. We are going to tell them through an event bus.
