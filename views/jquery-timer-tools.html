jQuery Timer Tools

Throttle and Debounce are handy tools for controlling the intensity of callback functions triggered by events. We have allready looked at debounce in 'mobileScrolling' project, but this time we are going to use a third party library with different tools.

jquery.timer-tools

https://github.com/lolmaus/jquery.timer-tools (1.2 KB).

Installed via bower (see bower.json).



Debounce

Execute a function only one time (even if we continously move mouse, event only fires once) within a time span specified in milliseconds.
There are two variants: 'execute at the begining of the time span' or 'execute at the end of the time span'

Throttle

Execute a function every n millisecond.
