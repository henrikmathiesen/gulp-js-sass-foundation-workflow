JS Build with Gulp and SASS

Goals

  • Learn about Gulp, following the Pluralsight course "JavaScript Build Automation With Gulp.js"
  • Bundle JavaScript
  • Minify JavaScript
  • Compile SASS to CSS
  • Minify CSS
  • We want to be able to debug the source with either source map or dev/build environment
  • Include Foundation and Font Awesome, set it up (SASS and JS)
  • Tie it all together with Gulp

Overview of Gulp API

gulp.task

Define the unique name for the task and a function to execute when task is run. If the task has dependencies, those will be handled (run in parallel) to aid the task.

gulp.task('name', [dependencies], function)

gulp.src

The first argument is called "glob" and is the path to the source files.

return gulp.src('./file/path/**/*.js', [options])

gulp.dest

To where do we write the file(s) when done.

return gulp.pipe(gulp.dest('./build/', [options]))

gulp.watch

Watch files and run a series of tasks (defined in an array) when any of those files changes. Alternative can switch the array for a callback function (that gets an event argument)

gulp.watch('./file/path/**/*.js', [options], ['task01', 'task02'])
